This Neovim configuration inherits all default LazyVim keybindings with minimal custom overrides.

## Custom Keymaps

The custom keymaps file is intentionally minimal, relying on LazyVim defaults:

```lua ~/workspace/source/nvim/lua/config/keymaps.lua
-- Keymaps are automatically loaded on the VeryLazy event
-- Default keymaps that are always set: https://github.com/LazyVim/LazyVim/blob/main/lua/lazyvim/config/keymaps.lua
-- Add any additional keymaps here
```

<Tip>
  Press `<leader>sk` in Neovim to search all available keybindings using Telescope.
</Tip>

## Plugin-Specific Keybindings

### Completion (nvim-cmp)

Custom completion keybindings from `tab-complete.lua`:

<CardGroup cols={2}>

<Card title="Tab Key" icon="arrow-right">
  **Behavior**: Auto-select first completion item if none selected, then confirm
  
  ```lua
  ["<Tab>"] = cmp.mapping(function(fallback)
    if cmp.visible() then
      local entry = cmp.get_selected_entry()
      if not entry then
        cmp.select_next_item({ behavior = cmp.SelectBehavior.Select })
      else
        cmp.confirm()
      end
    else
      fallback()
    end
  end, { "i", "s" })
  ```
</Card>

<Card title="Enter Key" icon="arrow-down">
  **Behavior**: Abort completion and insert newline (prevents accidental completion)
  
  ```lua
  ["<CR>"] = cmp.mapping(function(fallback)
    cmp.abort()
    fallback()
  end, { "i", "s", "c" })
  ```
</Card>

</CardGroup>

## Adding Custom Keybindings

To add your own keybindings, edit `~/workspace/source/nvim/lua/config/keymaps.lua`:

```lua
-- Example: Add custom keymap
local map = vim.keymap.set

-- Save file with Ctrl+S
map({ "n", "i", "v" }, "<C-s>", "<cmd>w<cr><esc>", { desc = "Save file" })

-- Clear search highlighting
map("n", "<leader>h", "<cmd>nohlsearch<cr>", { desc = "Clear search highlight" })
```

<Warning>
  Make sure your custom keybindings don't conflict with LazyVim defaults. Check existing mappings with `<leader>sk`.
</Warning>
