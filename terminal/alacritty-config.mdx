## Overview

Alacritty is a modern, GPU-accelerated terminal emulator written in Rust. This configuration provides a customized setup with enhanced visuals, fonts, and window behavior.

## Main Configuration

The primary configuration file is located at `~/.config/alacritty/alacritty.toml`.

### Theme Import

```toml alacritty.toml
general.import = [
    "~/.config/alacritty/themes/themes/coolnight.toml"
]
```

<Tip>
You can easily switch themes by changing the import path. See the [Themes](/terminal/themes) page for all 151 available themes.
</Tip>

### Environment Variables

```toml alacritty.toml
[env]
TERM = "xterm-256color"
```

Sets the terminal type to support 256 colors, ensuring proper color rendering in terminal applications.

## Window Settings

### Padding

```toml alacritty.toml
[window]
padding.x = 10
padding.y = 10
```

Adds 10 pixels of padding on all sides for a more comfortable viewing experience.

### Window Appearance

```toml alacritty.toml
decorations = "Buttonless"
opacity = 0.8
blur = true
startup_mode = "Maximized"
```

## Font Configuration

### Font Family

```toml alacritty.toml
[font]
normal.family = "0xProto Nerd Font Mono"
size = 16
```

**0xProto Nerd Font Mono** is used as the primary font, providing:
- Excellent readability for code
- Full Nerd Font icon support
- Monospaced design perfect for terminal work
- Professional appearance

<Tip>
Font size of **16** provides comfortable readability on most displays. Adjust based on your screen resolution and preferences.
</Tip>

For more details on font options and installation, see [Font Configuration](/terminal/fonts).

## Quick Customization

### Change Theme

1. Browse available themes in `~/.config/alacritty/themes/themes/`
2. Update the import path in `alacritty.toml`:

```toml
general.import = [
    "~/.config/alacritty/themes/themes/YOUR_THEME.toml"
]
```

### Adjust Window Opacity

```toml
[window]
opacity = 0.95  # More opaque
opacity = 0.7   # More transparent
```

### Modify Font Size

```toml
[font]
size = 14  # Smaller
size = 18  # Larger
```

## Advanced Settings

### Disable Blur (Better Performance)

```toml
[window]
blur = false
```

### Full Window Decorations

```toml
[window]
decorations = "Full"  # Show all title bar buttons
```

### Different Startup Modes

```toml
[window]
startup_mode = "Windowed"    # Normal window
startup_mode = "Fullscreen"  # Full screen
startup_mode = "Maximized"   # Maximized (default)
```
