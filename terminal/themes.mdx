## How to Switch Themes

### Method 1: Edit Configuration

1. Open your Alacritty config:
   ```bash
   vim ~/.config/alacritty/alacritty.toml
   ```

2. Change the import line:
   ```toml
   general.import = [
       "~/.config/alacritty/themes/themes/THEME_NAME.toml"
   ]
   ```

3. Save and the changes apply instantly!

<Note>
Alacritty automatically reloads configuration when the file changes - no need to restart.
</Note>

### Method 2: Quick Preview

Browse all themes in the directory:

```bash
ls ~/.config/alacritty/themes/themes/
```

## Tips for Choosing a Theme

<CardGroup cols={2}>
  <Card title="Consider Your Environment" icon="brightness">
    Use **dark themes** in low light, **light themes** in bright environments
  </Card>
  
  <Card title="Syntax Highlighting" icon="code">
    Ensure your theme works well with your editor's syntax colors
  </Card>
  
  <Card title="Eye Strain" icon="eye">
    Choose themes with appropriate contrast for extended sessions
  </Card>
  
  <Card title="Personal Preference" icon="heart">
    Try multiple themes to find what feels right for you
  </Card>
</CardGroup>

<Tip>
**Pro tip**: Keep a few favorite themes and rotate them based on time of day or mood. Many developers use a dark theme at night and light theme during the day.
</Tip>
